# 🎮 Minesweeper — Інтерактивна гра Сапер

Повна реалізація класичної гри **Sснапер** (Minesweeper) на **JavaScript** з красивим готичним інтерфейсом.

## 📋 Вимоги

- Браузер з підтримкою ES6
- Локальний файловий сервер (або просто відкрити `index.html`)

## 🚀 Запуск

1. Помістіть файли в папку:
   ```
   BesedinMaxym/
   ├── index.html
   ├── style.css
   ├── script.js
   └── FEATURES.md (додатково)
   ```

2. Відкрийте `index.html` в браузері:
   ```bash
   # Простий спосіб — двічі клікніть index.html
   # Або запустіть локальний сервер:
   python -m http.server 8000
   # Потім перейдіть на http://localhost:8000/BesedinMaxym/
   ```

## 🎮 Правила гри

1. **Ліве кліку** — розкрити клітинку
2. **Праве кліку** — встановити прапорець 🚩
3. **Перемога** — розкрийте всі безпечні клітинки (71 штук)
4. **Програш** — натисніть на міну 💣

## ⏱️ Механізм

- **Таймер** запускається при першому кліку
- **Прапорці** показують залишилися міни (10 всього)
- **Числа** показують кількість сусідніх мін (1–8)
- **Порожні клітинки** розкривають сусідів автоматично

## 🎨 Чемпіони інтерфейсу

- **Темна готична тема** 🖤
- **Паління на клітинках** при наведенні
- **Яскраві числа** з різними кольорами
- **Анімація модального вікна** для результату
- **Зміна лиця** на кнопці старту (💀 → 😎 / 😵)

## 📁 Структура файлів

### `index.html`
- Контейнер гри
- Панель зі счетчиками та кнопкою старту
- Поле гри (динамічно заповнюється)
- Модальне вікно для повідомлення про результат

### `style.css`
- CSS змінні для кольорів
- Стилі для всіх станів клітинок
- Grid-сітка для поля 9x9
- Анімації та переходи

### `script.js`
- **Логіка гри** (чисті функції):
  - `generateField()` — генерація поля
  - `openCell()` — вскриття клітинок
  - `toggleFlag()` — встановлення прапорців
  - `checkWinCondition()` — перевірка перемоги
  
- **UI функції**:
  - `renderBoard()` — відображення поля
  - `updateTimer()` — оновлення таймера
  - `updateFlagCounter()` — оновлення лічильника
  - `setupEventDelegation()` — налаштування подій

## 🔧 Як працює

### Генерація поля
```javascript
gameState = generateField(9, 9, 10);
// Створює 9x9 поле з 10 випадковими мінами
```

### Відкриття клітинки
```javascript
gameState = openCell(gameState, row, col);
// Якщо 0 сусідніх мін → рекурсивно відкриває сусідів
// Якщо міна → гра програна
```

### Делегування подій
```javascript
field.addEventListener('click', handleLeftClick);
field.addEventListener('contextmenu', (e) => {
  e.preventDefault();
  handleRightClick(e);
});
```

## 💡 Оптимізація

- ✅ **Чисті функції** — без побічних ефектів
